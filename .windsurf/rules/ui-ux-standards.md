# Apple Pro UI & Design Standards

Maintain the high-end professional aesthetic of the TPMS:

- **Visual Hierarchy**: Use `Sidebar + Main Detail` layouts for complex data (Cases, Clients).
- **Typography**: Strictly use `System-ui` (San Francisco) fonts.
- **Components**: Use `shadcn/ui` and `cult-ui` components. Customize with Tailwind `rounded-xl` to `rounded-3xl` for that premium look.
- **Interactions**:
  - Implement `backdrop-blur` for modals and sticky headers.
  - 44pt minimum hit targets for all interactive elements.
- **Data Density**: Balance white space with data density. Lawyers need to see lot of information but it must remain legible and organized.
